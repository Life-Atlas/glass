# Contributing to GLASS

GLASS is an open framework. Contributions welcome.

## How to Contribute

1. **Report gaslighting patterns** — Open an issue with a before/after example of AI overstating work
2. **Add user stories to the audit** — The audit script checks 8 stories. Add more for your domain
3. **Port to other tools** — Claude Code hook, GitHub Action, VS Code extension, Cursor integration
4. **Improve detection** — Better heuristics for mock data, dead code, branding leaks

## Pull Requests

- Keep changes small and focused
- Include a before/after example if adding a new detection
- Run the audit against a real codebase before submitting
- Use the GLASS scale in your PR description (what level is this PR at?)

## Code of Conduct

Be direct. Be honest. Don't gaslight.
